# Storyteller Assessment Tool

Organizational diagnostic for Chief Storyteller roles - 6 moments from the first 90 days that reveal whether an org is actually ready.

## Deployment

Auto-deploys to Netlify on push to `main` branch.

## Local Development

Just open `index.html` in a browser.

## Form Submissions

Form submissions appear in Netlify dashboard under Forms.

## Tech Stack

- Pure HTML/CSS/JavaScript (no dependencies)
- Netlify Forms for lead capture
- Mobile-first responsive design
